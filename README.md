# happyH-ANS-selftest

자율신경실조증 자가설문 웹페이지 🌿

병원에 오기 전, 환자와 보호자가 부담 없이 스마트폰으로 체크해볼 수 있는
**자율신경 밸런스 셀프 테스트**입니다. 이름·나이 등 개인정보 입력 없이,
전신 증상 / 뇌·신경 / 두피·피부 / 가슴 / 팔 / 비뇨기 / 소화기 / 다리
8개 영역 총 40개 문항을 체크하면 즉시 결과가 나옵니다.

## 결과 판정 기준

| 체크 개수 | 결과 |
|---|---|
| 3개 미만 | 자율신경실조증 가능성 낮음 → 안심하세요 |
| 3–5개 | 가능성 있음 |
| 6–10개 | 자율신경실조증 의심됨 → 전문의와 상담 필요 |
| 11개 이상 | 가능성 큼 → 즉시 전문 클리닉에서 검사·치료 시작 권장 |

## 특징

- 순수 HTML/CSS/JS 단일 파일(`index.html`)로 동작 — 별도 서버나 빌드 없이 바로 배포 가능
- 개인정보(이름, 나이, 연락처 등) 입력란 없음, 모든 계산은 브라우저 안에서만 처리
- 카테고리별 진행바, 체크 애니메이션, 결과에 따른 이모지 반응·컨페티 등 재미 요소 포함
- 결과 화면에서 영역별 체크 분포를 간단한 막대그래프로 확인 가능
- 사용자가 **동의 체크박스**를 선택했을 때만 "결과 보내기" 버튼이 활성화되며,
  실명 없이 체크 개수·등급만 메일(`mailto:`)로 전송할 수 있음 (전송 전 항상 사용자 확인 필요)
- 결과 전송 시 **확인용 별명**을 선택 입력할 수 있음 — 실명 대신 이니셜/별명 등을
  남기면, 병원에서 내원 시 어떤 결과와 매칭되는 분인지 구분하는 데 참고할 수 있음
- **"전체 결과 복사"** 버튼은 종이 초진문진표처럼 문항 40개 전체를
  ☑/☐ 로 표시한 상세 체크리스트(+ 별명·작성일·판정 결과)를 클립보드로
  복사함 — 병원 EMR의 문진 입력란에 그대로 붙여넣기 좋도록 설계됨
  ("결과 보내기" 메일은 용량 제한 때문에 요약본만 전송하고,
  전체 문항 내역은 복사 버튼으로 확인)

## 병원 EMR에 붙여넣기 (초진문진처럼 활용하기)

1. 환자가 테스트를 마치고 **"전체 결과 복사"** 버튼을 누르면 클립보드에
   아래와 같은 형식의 전체 문진 기록이 복사됩니다.

   ```
   [자율신경 밸런스 자가문진 결과]
   작성일: 2026-07-23
   확인용 별명: 파란고양이
   총 체크: 9 / 40
   결과 판정: 자율신경실조증 의심 — 증상이 여러 개 겹쳐 있어요...

   ▶ 주요 호소 증상 (체크된 항목)
   - 🌡️ 미열이 있는 것처럼 몸이 으슬으슬하다
   - 🧠 잠이 잘 오지 않는다
   ...

   ▶ 영역별 상세 체크리스트
   [전신 증상] (1/6)
   ☑ 미열이 있는 것처럼 몸이 으슬으슬하다
   ☐ 최근 3개월 사이 3kg 이상 체중이 변했다
   ...
   ```

2. 환자가 내원했을 때, 이 텍스트를 카카오톡/문자로 미리 받아두었다가
   접수 시 **EMR의 초진문진/주소증 입력란에 그대로 붙여넣기**만 하면
   됩니다. 문항 전체가 체크/미체크로 남아있어 종이 문진표를 그대로
   옮긴 것과 같은 효과를 냅니다.
3. "결과 보내기"(메일) 는 이 전체 목록 대신 **요약본**(별명·총점·영역별
   개수)만 보내도록 되어 있습니다 — 일부 메일 앱/기기에서 너무 긴
   `mailto:` 링크가 깨지는 것을 막기 위한 설계이니, 전체 문항 내역이
   필요하면 항상 "전체 결과 복사"를 사용해주세요.

## 병원 수신 메일 주소 변경하기

`index.html` 안의 아래 줄에서 이메일 주소만 바꾸면 됩니다.

```js
const HOSPITAL_EMAIL = 'nothing4ukim@gmail.com';
```

## 로컬에서 열어보기

별도 설치 없이 `index.html` 파일을 브라우저로 열기만 하면 바로 사용할 수 있습니다.

## GitHub Pages로 배포해서 누구나 접속하게 하기

1. GitHub 저장소 페이지에서 **Settings → Pages** 로 이동
2. **Build and deployment → Source** 를 `Deploy from a branch` 로 설정
3. Branch를 `main`(또는 배포할 기본 브랜치), 폴더는 `/ (root)` 선택 후 저장
4. 잠시 후 `https://<사용자명>.github.io/<저장소명>/` 주소로 접속 가능

배포 후 이 링크를 환자, 보호자, 주변 분들께 카카오톡이나 문자로 공유하면
누구나 별도 설치 없이 바로 자가설문을 진행할 수 있습니다.

## 안내

이 테스트는 참고용 자가설문이며, 의학적 진단을 대신하지 않습니다.
결과와 무관하게 증상이 지속되면 전문의와 상담하시길 권해드립니다.
